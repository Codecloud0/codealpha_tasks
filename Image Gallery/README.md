# 📸 Image Gallery

A responsive image gallery built with HTML, CSS, and JavaScript, featuring category filters, live search, and a full lightbox viewer.

## Features
- Responsive grid layout (auto-adjusts for desktop, tablet, mobile)
- Category filters: All, Nature, City, Abstract, Space
- Live search by title or category
- Lightbox view with next/prev navigation
- Keyboard support (← → arrows, Esc to close)
- Touch/swipe support on mobile
- Dark/light theme toggle (persists via localStorage)
- Smooth hover effects and transitions

## Tech Stack
- HTML5
- CSS3 (Flexbox, Grid, custom properties)
- Vanilla JavaScript (no frameworks)

## How to Run
1. Clone this repository
   \`\`\`bash
   git clone https://github.com/Codecloud0/image-gallery.git
   \`\`\`
2. Open \`index.html\` in your browser, or use a local server (e.g. VS Code Live Server)
