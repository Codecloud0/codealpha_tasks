# CodeAlpha Internship Tasks

This repository contains front-end web development projects completed as part of the CodeAlpha internship program.

## 📁 Projects

### 1. [Image Gallery](./Image Gallery)
A responsive image gallery with category filters, live search, and a full-featured lightbox viewer.

**Key Features:**
- Responsive grid layout
- Category filters (Nature, City, Abstract, Space)
- Live search by title/category
- Lightbox with next/prev navigation and keyboard support
- Dark/light theme toggle

**Tech:** HTML, CSS, JavaScript

---

### 2. [Music Player](./Music Player)
A modern glassmorphism-style music player with full playback controls and a playlist.

**Key Features:**
- Play, pause, next, previous controls
- Progress bar with seek functionality
- Volume control
- Shuffle and repeat modes
- Playlist with autoplay

**Tech:** HTML, CSS, JavaScript, HTML5 Audio API

---

## 🛠️ How to Run

Each project is self-contained. To run any project:

1. Navigate into the project folder
2. Open `index.html` in your browser
   - Or use a local server (e.g. VS Code Live Server extension) for the best experience
