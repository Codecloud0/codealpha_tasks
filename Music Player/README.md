# 🎵 Music Player

A modern glassmorphism-style music player built with HTML, CSS, and JavaScript.

## Features
- Play, pause, next, previous controls
- Live progress bar with seek functionality
- Volume control with mute icon states
- Song title, artist, and duration display
- Shuffle and repeat (off / repeat all / repeat one) modes
- Full playlist with click-to-play
- Autoplay to next track
- Spinning album art animation while playing

## Tech Stack
- HTML5 Audio API
- CSS3 (glassmorphism, custom range sliders)
- Vanilla JavaScript

## How to Run
1. Clone this repository
   \`\`\`bash
   git clone https://github.com/Codecloud0/music-player.git
   \`\`\`
2. Open \`index.html\` in your browser, or use a local server
